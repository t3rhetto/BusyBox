# BusyBox

> A "Look Busy" Toolbox for macOS
> 
> Pretend you're hard at work with this fake terminal tool

[中文](#中文) | [English](#english)

---

## English

### Overview

BusyBox is a macOS native application that displays a fake terminal continuously outputting AI coding content, making it appear like you're hard at work with Cursor, Copilot, or Claude.

### Features

- 💻 **Fake Terminal**: Realistic terminal interface with dark background and monospace font
- 🤖 **AI Coding Output**: Continuously generates realistic code output (React, Python, Rust, etc.)
- 🪟 **Multi-Window**: Open multiple terminal windows simultaneously
- ⌨️ **Keyboard Shortcut**: `Cmd + N` to open new terminal window
- 🎨 **macOS Native**: Built with Swift and SwiftUI

### Installation

#### Option 1: Download Release

1. Go to [Releases](https://github.com/t3rhetto/BusyBox/releases)
2. Download `BusyBox-macOS-universal.zip`
3. Unzip and drag `BusyBox.app` to your Applications folder
4. Right-click and select "Open" (first time only, due to Gatekeeper)

#### Option 2: Build from Source

```bash
git clone https://github.com/t3rhetto/BusyBox.git
cd BusyBox
swift run
```

### Requirements

- macOS 14.0+ (Sonoma)

### Usage

1. Launch BusyBox
2. A fake terminal window will appear with continuous AI coding output
3. Press `Cmd + N` to open more terminal windows
4. Look busy! 😎

## 中文

### 概述

BusyBox 是一款 macOS 原生应用，显示一个假终端并持续输出 AI 编程内容，让老板以为你在用 Cursor、Copilot 或 Claude 认真写代码。

### 功能特点

- 💻 **假终端**: 逼真的终端界面，深色背景 + 等宽字体
- 🤖 **AI 编程输出**: 持续生成逼真的代码输出（React、Python、Rust 等）
- 🪟 **多窗口**: 同时打开多个终端窗口
- ⌨️ **快捷键**: `Cmd + N` 快速打开新终端窗口
- 🎨 **原生体验**: 使用 Swift 和 SwiftUI 构建

### 安装方式

#### 方式一：下载发布版本

1. 前往 [Releases](https://github.com/t3rhetto/BusyBox/releases)
2. 下载 `BusyBox-macOS-universal.zip`
3. 解压后将 `BusyBox.app` 拖入应用程序文件夹
4. 首次打开需右键选择"打开"

#### 方式二：从源码编译

```bash
git clone https://github.com/t3rhetto/BusyBox.git
cd BusyBox
swift run
```

### 系统要求

- macOS 14.0+ (Sonoma)

### 使用方法

1. 启动 BusyBox
2. 假终端窗口会自动出现并持续输出 AI 编程内容
3. 按 `Cmd + N` 打开更多终端窗口
4. 假装很忙！😎
